## 1. Project Overview

```md
This is a personal portfolio website built using static HTML, CSS, and JavaScript.

The goal is to present a clean, modern portfolio focused on data engineering and analytics work, highlighting projects, impact, and technical ownership.

The site is hosted on GitHub Pages and must remain fully static.
```

---

## 2. Technical Constraints (VERY IMPORTANT)

```md
- Must remain static (no backend)
- No frameworks (no React, Vue, etc.)
- No Jekyll
- Must work on GitHub Pages
- Keep dependencies minimal
```

---

## 3. Design Principles

```md
- Clean and modern (data/finance style)
- Light mode (primary)
- Minimal but structured
- Focus on readability and spacing
- Prioritise content over visuals
```

---

## 4. Architecture Rules

```md
- Modular structure (clear sections)
- Reusable project layout
- No inline styles
- Clean, readable HTML
- Separate CSS files where appropriate
```

---

## 5. Content Rules

```md
- Focus on impact and results
- Use quantified achievements where possible
- Avoid generic descriptions
- Prioritise best projects first
```

---

## 6. Project Prioritisation

```md
Featured projects should appear first, including:
- Movie Recommendation System
- NLP Sentiment Analysis
- Regulatory Reporting System
- Mortgage Portfolio Integrations
- Treasury System Integration
- Pipeline Optimisation
```

---

## 7. UX Rules

```md
- No tab-based navigation for projects
- Use scroll-based layout or cards
- Ensure content is easily scannable
- Keep interactions minimal and smooth
```
