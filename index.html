Project 1 - IMDb NLP Sentiment Analysis

Goal

The goal of this project was to harness the power of Natural Language Processing (NLP) to gain insights from IMDb movie reviews. It aimed to uncover sentiments within textual data and provide valuable information for decision-making.

Work Done

I retrieved extensive movie review data from the IMDb database using Google BigQuery. This involved crafting SQL queries, performing complex joins, and fine-tuning the dataset to ensure it was ready for analysis.

To gain a comprehensive understanding of the dataset, I employed Looker, a powerful data exploration and visualization tool. This facilitated the creation of informative dashboards that visually represented key trends and patterns.

Leveraging Google Colab, I seamlessly integrated the data into a Pandas DataFrame, allowing for further manipulation and analysis. This transition streamlined the process and enhanced data accessibility.

Utilizing NLP libraries, I applied Count Vectorization to the textual content, enabling the classification of reviews as positive or negative sentiment. This step was pivotal in extracting actionable insights from unstructured text data.

Additionally, I extended the project by creating a Convolutional Neural Network (CNN) model for sentiment prediction. This model utilized the textual data to make accurate sentiment predictions, enhancing the depth of analysis and predictive capabilities.

Outcome

The project showcased my expertise in NLP, data preprocessing, visualization, and machine learning. It allowed for valuable insights to be derived from IMDb movie reviews, which can aid in decision-making processes. The best performing model could classify 87% of movie reviews accurately.

Project 2 - Movie Recommendation System

Goal:

In this project, I set out to create a personalized movie recommendation system, aiming to enhance the cinematic experiences of movie enthusiasts. Leveraging cutting-edge data science and machine learning techniques, I ventured into the world of recommendation algorithms to deliver tailored movie suggestions.

Work Done:

I began by implementing a foundational recommendation model using Singular Value Decomposition (SVD). This method allowed me to establish a benchmark for comparison in later stages of the project.

Prior to diving into modeling, I conducted extensive exploratory data analysis (EDA) to comprehend the underlying patterns in the movie dataset. This analysis served as a crucial foundation for the recommendation algorithms.

In addition to SVD, I implemented machine learning algorithms, including Regression, Support Vector Regression (SVR), and Random Forest, to further refine the recommendation engine. These algorithms added depth and accuracy to the suggestions, improving on the base line loss by 36%.

To push the boundaries of recommendation precision, I experimented with PyTorch and neural networks. This innovative approach allowed for more intricate modeling and improved personalization.

Outcome:

The project aimed to create a personalized movie recommendation system, enhancing the cinematic experiences of users. The use of various recommendation algorithms and machine learning techniques improved the accuracy of movie suggestions. The project achieved a 36% improvement in loss compared to the baseline model.
